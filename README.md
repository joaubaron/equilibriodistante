
# Equilíbrio Distante — PWA

PWA em tela cheia inspirado no álbum :contentReference[oaicite:0]{index=0} de :contentReference[oaicite:1]{index=1}.

Navegue pelos encartes deslizando horizontalmente enquanto ouve as músicas do álbum.

---

## ✨ Recursos

- 📀 Encarte horizontal fullscreen
- 🎵 Reprodução integrada das faixas
- ⏯ Reprodução contínua automática
- 📱 Instalável como app (PWA)
- 📶 Funciona offline
- 🎧 Media Session API
- 🔋 Wake Lock

---

## 📁 Estrutura

```text
/equilibrio
├── index.html
├── manifest.json
├── sw.js
├── /encartes
└── /mp3
```

---

## 🖼 Encartes

Coloque as imagens em:

```text
/encartes
```

Exemplo:

```text
encarte1.webp
encarte2.webp
encarte3.webp
```

---

## 🎵 Músicas

Coloque os MP3 em:

```text
/mp3
```

Exemplo:

```text
01-Gente.mp3
02-Strani Amori.mp3
```
